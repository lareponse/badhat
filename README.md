## European Accessibility Act (EAA)

## EN 301 549
EN 301 549 is the European Standard for the accessibility of ICT (Information and Communication Technology) products and services. It sets detailed technical requirements to comply with the Web Content Accessibility Guidelines (WCAG) and other accessibility standards.

Web Developers and Software Providers must ensure websites and apps are WCAG 2.1 AA compliant.
