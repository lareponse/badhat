# Technologies Assistives par Type de Handicap

## 1. Pour les Personnes Aveugles

### Technologies Principales
- **Lecteurs d'écran**
  - Logiciels : NVDA (gratuit), JAWS (payant), VoiceOver (intégré iOS/Mac), TalkBack (Android)
  - Coût : De gratuit à environ 1000€ pour une licence JAWS
  - Formation disponible via les associations locales d'aide aux personnes déficientes visuelles

- **Afficheurs braille**
  - Modèles : Brailliant, Focus, BrailleNote
  - Prix : Entre 2000€ et 8000€ selon le modèle
  - Possibilité d'aide au financement via la MDPH

- **Applications mobiles spécialisées**
  - Be My Eyes : Assistance visuelle en direct par des bénévoles (gratuit)
  - Seeing AI : Description d'images et lecture de texte (gratuit)
  - Sullivan+ : Navigation assistée par IA

### Innovations Récentes
- **Lunettes intelligentes**
  - OrCam MyEye : Lecture de texte et reconnaissance faciale
  - Coût : environ 4000€
  - Prise en charge partielle possible par la sécurité sociale

## 2. Pour les Personnes Malvoyantes

### Logiciels et Paramètres
- **Solutions d'agrandissement**
  - ZoomText (payant, ~700€)
  - MAGic (payant, ~600€)
  - Loupe Windows (gratuit)
  - Zoom macOS (gratuit)

- **Paramètres d'accessibilité natifs**
  - Options de contraste élevé
  - Filtres de couleur personnalisables
  - Curseurs agrandis
  - Thèmes adaptés

### Options des Navigateurs
- **Extensions gratuites**
  - Dark Reader pour le mode sombre
  - Zoom Page WE pour le zoom personnalisé
  - High Contrast pour l'amélioration du contraste

## 3. Pour les Personnes Sourdes

### Communication
- **Sous-titrage en direct**
  - Google Live Transcribe (gratuit)
  - Ava (abonnement mensuel)
  - Microsoft Translator (gratuit)

- **Visioconférence accessible**
  - Zoom avec sous-titrage automatique
  - Teams avec transcription en temps réel
  - Google Meet avec sous-titrage

### Équipement Domestique
- **Systèmes d'alerte**
  - Sonnettes avec flash lumineux
  - Réveils vibrants
  - Détecteurs de fumée avec signal visuel
  - Prix moyen : 100-300€ par dispositif

## 4. Pour les Troubles Cognitifs et du Langage

### Outils de Communication
- **Applications AAC**
  - Proloquo2Go (payant, ~250€)
  - Let Me Talk (gratuit)
  - Grid 3 (payant, ~700€)

### Organisation et Planification
- **Applications de gestion du temps**
  - TimeTimer (visuel)
  - Agenda visuel personnalisé
  - Rappels avec pictogrammes

### Aide à la Lecture
- **Outils de simplification**
  - Lirec (textes adaptés)
  - Bienvenu (textes FALC)
  - Adapte ton texte (reformulation)

## 5. Pour les Troubles Psychomoteurs

### Dispositifs d'Accès
- **Solutions alternatives**
  - Trackball (50-200€)
  - Joystick adapté (100-300€)
  - Contacteur unique (50-150€)

### Domotique Adaptée
- **Contrôle environnemental**
  - Commande vocale compatible (Alexa, Google Home)
  - Interrupteurs adaptés
  - Télécommandes universelles simplifiées

## Ressources et Accompagnement

### Financement
- Dossier MDPH pour aide à l'acquisition
- PCH (Prestation de Compensation du Handicap)
- Possibilités de prêt de matériel pour essai

### Formation
- Centres de réadaptation spécialisés
- Associations locales d'entraide
- Tutoriels en ligne gratuits
- Sessions de formation personnalisées

### Compatibilité
- Vérification de la compatibilité entre dispositifs
- Tests préalables recommandés
- Consultation d'ergothérapeutes pour le choix optimal

Ce document respecte les normes d'accessibilité WCAG 2.1 et peut être consulté avec les technologies d'assistance courantes. Les retours d'expérience d'utilisateurs sont régulièrement intégrés pour maintenir sa pertinence.
